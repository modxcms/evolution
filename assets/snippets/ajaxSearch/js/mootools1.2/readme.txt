Readme text file for js/mootools1.2

The mootools.js file of the js/mootools1.2 directory contains 

1/ the following mootools1.2 core components:

- Core
- Native
- Class
- Elements (except Elements.dimension)
- DomReady
- Fx
- Request

For more informations see: http://mootools.net/core version 1.2.1

2/ and from MooTools 1.2 More Builder

the following plugin:
- Fx.Slide

For more informations see: http://mootools.net/more version 1.2.1

This package is a YUI compressed package.
These are the minimum components/plugin needed by js/ajaxSearch-mootools1.2.js file
used by AjaxSearch with the ajax mode (&ajaxSearch=`1`)
